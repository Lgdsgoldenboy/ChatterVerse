# **App Name**: ChatterVerse

## Core Features:

- User Registration & Authentication: Allow users to register and authenticate using email or social media accounts (Google, Facebook).
- Markdown-based Content Creation: Provide a rich text editor that supports Markdown for writing and publishing content with images and videos.
- Personalized Content Feed: Create a personalized content feed based on user interests and reading history.
- Content Discovery: Enable browsing, searching, and filtering of content by categories and tags.
- Social Interaction: Implement social features like commenting, liking, and bookmarking for user interaction.
- Content Analytics: Provide analytics for content creators to track views, likes, comments, and bookmarks.
- Content Moderation Tool: Leverage AI to moderate generated content (identify problematic posts, and content policy violations). An LLM reasons if content should be flagged for review by human moderators.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to convey a sense of calmness and reliability, fitting for a text-focused app.
- Background color: Light grey (#F0F4F7), almost white, creating a clean, readable backdrop for content.
- Accent color: Light Purple (#B39DDB) to add a touch of creativity and modernity to interactive elements.
- Headline font: 'Belleza', a humanist sans-serif, gives a modern artistic style for headlines.
- Body font: 'Alegreya' A humanist serif with an elegant, intellectual, contemporary feel
- Simple, minimalist icons to represent categories, tags, and social actions.
- Clean and structured layout with clear content hierarchy for easy reading and navigation.
- Subtle transitions and animations for interactive elements like likes and comments.
